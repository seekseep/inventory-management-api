# シードデータ ストーリー

メンズアパレルセレクトショップ「OUTLINE」の5年間（2020年4月〜2025年3月）。

## 商品数の目安

| 指標 | 説明 |
|---|---|
| スタイル | 1つのデザイン（例: スリムデニム） |
| SKU | スタイル × カラー × サイズ（例: スリムデニム/ブ��ック/M） |
| カラー展開 | 定番: 2〜4色、シーズン: 2〜3色、限定: 1〜2色 |
| サイズ展開 | S / M / L / XL の4サイズ（アクセサリーはFREE） |

| フェーズ | スタイル数 | SKU数（概算） |
|---|---|---|
| 開業時 | 20〜30 | 200〜350 |
| 成長期 | 50〜80 | 500〜900 |
| 衰退期 | 30〜40 | 300〜450 |
| 回復〜安定期 | 60〜80 | 600〜900 |
| 次の成長期 | 100〜120 | 1,000〜1,400 |

## 安全在庫の考え方

| 商品タイプ | 安全在庫/SKU/拠点 | 理由 |
|---|---|---|
| staple (定番) | 3〜5個 | 常に欠品を防ぎたい。再仕入れ可能 |
| seasonal (シーズン) | 1〜2個 | シーズン中は切らしたくないが、余りすぎも困る |
| limited (限定) | 0個 | 売り切り前提。安全在庫は設定しない |

安全在庫を下回ったら仕入れトリガーとなる想定。

## フェーズ詳細

### Year1 前半 (2020/4〜2020/9) — 初期の停滞

**状況**: 渋谷の裏通りに小さなメンズセレクトショップをオープン。SNSフォロワー200人からのスタート。

- **拠点**: 本店倉庫 + 渋谷店
- **商品**: 定番6スタイル(67SKU) + SS2020 8スタイル(~70SKU) = 約137SKU
- **トランザクション**:
  - 仕入れ: 月2回、1回あたり30〜50点
  - 移動: 倉庫→渋谷店 週1回、20〜30点
  - 販売: 1日3〜5点（週20〜35点）
  - 廃棄: なし
- **棚卸し**: 月末に各拠点1回
- **安全在庫**: 定番3個/SKU、シーズン1個/SKU
- **特徴**: 在庫回転率が低い。AW2020の仕入れ量を慎重に判断

### Year1 後半〜Year2 前半 (2020/10〜2021/9) — 最初の成長

**状況**: インスタでスリムデニムの着こなし投稿がバズる。来店客が3倍に。新宿に2号店出店。

- **拠点**: +新宿店（Y2Q1オープン）
- **商品**: 定番8スタイル(89SKU) + AW2020〜SS2021 + 1周年限定 = 約350〜400SKU
- **トランザクション**:
  - 仕入れ: 月3〜4回、1回あたり80〜120点
  - 移動: 倉庫→2店舗 週3回、各30〜50点
  - 販売: 1日15〜25点（2店舗合計）
  - 廃棄: ほぼなし
- **棚卸し**: 月末 × 3��点
- **安全在庫**: 定番5個/SKU、シーズン2個/SKU に引き上げ
- **特徴**: 1周年Tシャツが2日で完売。人気SKU（デニム/M/インディゴ）は欠品頻発

### Year2 後半〜Year3 前半 (2021/10〜2022/9) — 1度目の衰退

**状況**: 駅前に大手セレクトショップが出店。新宿店の客足が半減。撤退を決断。

- **拠点**: 新宿店を Y3Q1 に閉鎖
- **商品**: AW2021がセール→廃番。SS2022は仕入れ絞る = 約250〜300SKU
- **トランザクション**:
  - 仕入れ: 月1回、1回あたり30〜40点に縮小
  - 移動: 新宿店閉鎖時に一括200点以上を倉庫に戻���
  - 販売: 1日5〜8点に減少。セール期間は1日15〜20点
  - 廃棄: AW2021売れ残り約50点を廃棄
- **棚卸し**: 月末。閉店棚卸しで差異発覚（5点ロス）
- **安全在庫**: 定番3個/SKUに引き下げ、シーズン1個/SKU
- **特徴**: セール(on_sale)→廃番(discontinued)の遷移が大量発生

### Year3 後半〜Year4 前半 (2022/10〜2023/9) — 回復

**状況**: EC販売を開始。アーティストコラボパーカーがSNSで1万RTされ話題に。

- **拠点**: +ECストア（Y3Q3）
- **商品**: 定番9スタイル(101SKU) + AW2022〜SS2023 + コラボ = 約450〜500SKU
- **トランザクション**:
  - 仕入れ: 月2〜3回、1回あたり60〜100点に回復
  - 移動: 倉庫→渋谷店 週2回 + 倉庫→EC出荷 毎日5〜15点
  - 販売: 店舗 1日8〜12点 + EC 1日10〜20点
  - 廃棄: 月5〜10点（検品不合格品）
- **棚卸し**: 月末 × 3拠点
- **安全在庫**: 定番4個/SKU、EC分は倉庫で+2個バッファ
- **特徴**: EC比率が50%超に。コラボパーカー4SKU全て即日完売

### Year4 後半 (2023/10〜2024/3) — 安定

**状況**: 実店舗+ECの2チャネル体制が確立。仕入れ・販売が予測可能に。

- **拠点**: 本店倉庫 + 渋谷店 + ECストア（3拠点）
- **商品**: 定番10スタイル(101SKU) + AW2023 = 約600〜700SKU
- **トランザクション**:
  - 仕入れ: 月2回、計画的に100〜130点
  - 移動: 倉庫→店舗 週2回 + EC出荷 毎日10〜20点
  - 販売: 店舗 1日10〜15点 + EC 1日15〜20点
  - 廃棄: 月3〜5点
- **棚卸し**: 月末 × 3拠点（差異1%未満）
- **安全在庫**: 最適化済み。定番5個/SKU、シーズン2個/SKU
- **特徴**: 安全在庫割れが月2〜3回程度に抑制。予測精度向上

### Year5 (2024/4〜2025/3) — 次の成長

**状況**: 物流センターを増設。横浜に3号店をオープン。月商が過去最高を更新。

- **拠点**: +物流センター（Q1）+横浜店（Q2）→ 計5拠点
- **商品**: 定番10スタイル + SS2024/AW2024 + コラボ2件 = 約1,000〜1,400SKU
- **トランザクション**:
  - 仕入れ: 月4〜5回、1回あたり150〜250点。2倉庫に分散
  - 移動: 拠点間移動が複雑化。物流センター→各店舗、倉庫間移動も発生
  - 販売: 店舗2店で1日20〜30点 + EC 1日20〜30点
  - 廃棄: 月5〜10点
- **棚卸し**: 月末 × 5拠点
- **安全在庫**: 拠点別に最適化。店舗3個/SKU、倉庫5個/SKU
- **特徴**: デザイナーコラボMA-1(¥45,000)が即完売で話題。5周年フーディは全拠点で販売
